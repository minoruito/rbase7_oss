# RBASE7 OSS

業務アプリケーションを構築する際に必要となる基盤機能を備えた **基盤アプリケーション** です。認証・権限・ワークフロー・通知の枠組みを提供し、その上にドメイン固有の機能を載せる想定で設計されています。

## 主な機能

### ログイン管理

- **ローカルアカウント** によるログイン
- **SAML 2.0** における **サービスプロバイダ（SP）** としてのシングルサインオン

### 権限管理

- 権限（ロール）単位で、各機能の **利用可否** を設定可能

### ワークフロー

- 自由度の高い **ワークフローステータス** を定義可能
- ステータス変更のたびに **独自ロジックの実行** が可能

### メールテンプレート

- **メールテンプレート** の作成・管理
- ワークフローと連携し、状態遷移に応じた **メール通知** を実現可能

### 帳票テンプレート

- **帳票テンプレート** の作成・管理
- ThinReportのテンプレートファイルと編集クラスによって利便性の高い帳票作成を支援

### システム設定

- 各種システム設定をUIで編集

### プラグイン設定

- RBASE7_OSSは、フレームワークとしても機能し、新規クラスの追加や既存クラスのオーバーライドによって、機能追加や変更を行うことができます．

## 技術スタック（参考）

- Ruby **3.1.2**
- Rails **~> 7.0**
- MySQL（`mysql2`）

## セットアップの目安

詳細は環境に合わせて調整してください。

1. 依存関係のインストール: `bundle install`
2. データベース設定: `config/database.yml.sample` を参考に `config/database.yml` を用意
3. データベースの作成・マイグレーション: `bin/rails db:create db:migrate` など
4. 秘密情報・鍵類はリポジトリに含めず、環境変数やデプロイ先で設定すること（例: `lib/tasks/rbase.rake` や SAML 関連の `CHANGE_ME` プレースホルダを本番値に置き換え）
